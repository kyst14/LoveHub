# 💗 LoveHub

LoveHub is a personal web project that brings together a collection of small
interactive websites created as surprise experiences for someone special.

Each page represents a different moment or occasion — such as Valentine’s Day,
birthday, anniversary, and other meaningful dates — designed as a playful and
emotional journey.

---

## ✨ Purpose

The goal of this project is to turn simple web pages into memorable experiences
through:

- 💌 Personalized content
- 🎁 Hidden surprises and interactions
- 🎨 Creative and themed designs
- ❤️ Emotional storytelling through UI

LoveHub is not just a set of websites — it’s a digital space for creating shared
memories.

---

## ⚙️ Tech Stack

- ⚛️ Next.js
- 🎨 Tailwind CSS
- 🚀 Deployed on Vercel
- 📡 Optional event tracking via Discord webhooks

---

## 🔔 Events System (Optional)

The project can send interaction events (with consent) to a Discord channel
using webhooks, such as:

- Page opened
- Button clicked
- Surprise revealed

This is used only to enhance the experience and track interactions in real time.

---

## 💡 Features Ideas

- Animated transitions between pages
- Countdown timers for special dates
- Interactive surprise buttons
- Progress-based storytelling

---

## 🚀 Deployment

This project is optimized for deployment on Vercel:

```bash
npm install
npm run build
npm run start
```

Or deploy directly via the Vercel dashboard.

## ❤️ Philosophy

LoveHub is built around the idea that small digital experiences can carry real
emotion. Every interaction is designed to feel personal, thoughtful, and unique.
